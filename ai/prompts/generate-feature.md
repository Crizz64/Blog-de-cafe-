# Prompt: Generar Feature

## Uso
Usa este prompt cuando necesites que la IA implemente una nueva funcionalidad.

## Template

```
Contexto del proyecto:
- Blog de Café, sitio estático HTML/CSS/JS vanilla
- Sin frameworks ni bundler
- Stack: Firebase/Supabase para datos, Netlify para hosting
- Convenciones: ver ai/conventions.md

Feature a implementar: [DESCRIPCIÓN]

Spec de referencia: specs/[NOMBRE-SPEC].md

Archivos a modificar/crear:
- [ARCHIVO 1]
- [ARCHIVO 2]

Restricciones:
- No introducir nuevas dependencias
- Mantener compatibilidad con el código existente en [ARCHIVO]
- [OTRAS RESTRICCIONES]

Entrega:
1. Código completo listo para copiar
2. Instrucciones de integración
3. Qué testear manualmente
```
